Essa tag é um titulo personalizado

Herarquia de titulos vai do H1 ao H6

A tag de cima (hr) faz essa linha aparecer, ela é uma tag auto fechada e que pode ser editada em css, largura, cor e etc

O botão pode ter 3 tipos, o simples, o reset e o submit

Botão também possui onclick que indica algo quando clicado

Os botões também podem ser editados

A tag src serve para direcionar onde esta o caminho da imagem

A tag title serve para indicar qual titulo vai aparecer na imagem quando o mouse passar por cima

A tag alt serve para quando a imagem não puder ser carregada essa descrição apareça

As tags width e height definem a largura e a altura da imagem


A tag br serve para pular linha

FORMULARIO PESSOAL









A tag action indica para onde os dados digitados vão ir

A tag method indica qual metodo vamos usar o GET um metodo para informações publicas, POST um metodo para informações privadas

A tag label indica o que será colocado no campo

A tag input abre um retangulo para escrevermos dentro dele

As tags name, value e type indicam o nome da variavel, o valor que ela possui e o tipo dela

As tags max, min e step indicam, max=o maximo que aquela variavel chega, min=o minimo que chega e step é de quanto em quanto aquela variavel vai


FORMULARIO ESCOLHA

Quando o type for igual a radio vai abrir o campo de escolha entre uma ou outra opção

Em cada um dos inputs precisa estar o mesmo valor na variavel name


FORMULARIO CHECKBOX

Quando o type for igual a checkbox vai abrir quadrados onde voce pode marcar mais de uma opção

Em cada um dos inputs precisa estar o mesmo valor na variavel name
